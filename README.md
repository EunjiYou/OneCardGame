# OneCardGame
원카드 게임을 구현한 프로젝트입니다. 
<br/><br/>
# 세부 설명
- 개발기간 : 7일
- 사용 언어 : C#
- 사용 툴 : Visual Studio
<br/><br/><br/>
# 주요 특징
- 클래스 라이브러리를 사용하여 Console 프로젝트에서 WinForm 프로젝트 두 환경에서 쉽게 게임을 구현
- 특수 카드 능력 구현을 위한 Strategy Pattern 사용
<br/><br/><br/>
